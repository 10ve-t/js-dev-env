WEBフロントエンド開発キット
====

## Description
WEBフロントエンド開発のタスクをまとめたリポジトリ。gulpタスクを、npm scriptsから呼び出すように設計。

## You can use
- React
- Redux
- pug
- scss
- watch

## Install
- npm 3.10.3〜
- ``` npm install -g gulp ```

## Usage or Install
1. ``` git clone https://github.com/tatsuyoshi1225/js-dev-env.git ```
2. ``` cd js-dev-env ```
3. ``` npm install  ```
4. ``` npm run ```
